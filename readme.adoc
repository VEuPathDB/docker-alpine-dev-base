= Alpine Dev Base

link:https://hub.docker.com/r/veupathdb/alpine-dev-base[On Dockerhub]

Alpine linux container with Oracle Instant Client, OpenJDK,
and Maven installed.

Intended to be used as a build container for java based
services.
